FRONTEND SIDE
1. Install node JS and Mongo DB.
2. Download the code from the github.
3. Reinstall the react-scripts by the command: npm install react-scripts
3. Run the code by: npm start
4. You can access the site in this URL http://localhost:3000/


Note
- When editing the front end, you can edit the Home.js (src\pages)
- Home.js is where you also connect to localhost/3001 from the database.
